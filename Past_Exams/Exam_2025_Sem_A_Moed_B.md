## שאלה מספר 3 (20 נק')
עליכם להדגים (ולהסביר תוך כדי) פיתוח TDD (Test-Driven Development) עבור מערכת המלצת סרטים. על המימוש להיות ב-++C. עליכם להראות את תהליך ה-TDD ולממש 3 Unit Tests מסוגים שונים (למשל ב-Google Test). המערכת היא תוכנית CLI התומכת בפקודות הבאות:
*   `add [userid] [movieid1]...` : שיוך משתמש לסרטים שצפה בהם, נשמר לקובץ.
*   `recommend [userid] [movieid]` : מחזיר עד 10 המלצות.
*   `help` : מדפיס את רשימת הפקודות.

## שאלה מספר 4 (30 נק', שני סעיפים)
**א. (10 נק')** נתון הקוד הבא ב-++C:
```cpp
#include <unistd.h>
#include <iostream>
#include <sys/wait.h>
using namespace std;
int main() {
    int i=1;
    while((i % 3) != 0) {
        cout << i++ << endl;
        int foo = fork();
        if (foo != 0){
            wait(NULL);
            cout << i << endl;
        }
    }
    return 0;
}
```
כתבו את כל הפלטים האפשריים המדויקים, או הסבירו שגיאות קומפילציה/ריצה.

**ב. (20 נק')** כתבו קוד ב-Android (תקני ומלא, עם חלוקה לקבצים) המציג אפליקציה עם מסך יחיד: תיבת עריכת טקסט, שדה הצגת טקסט, וכפתור. לחיצה על הכפתור מעדכנת את שדה התצוגה. חובה לממש בעזרת MVVM בשילוב `LiveData` ו-Repository.
